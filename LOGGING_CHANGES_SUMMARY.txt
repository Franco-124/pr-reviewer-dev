═══════════════════════════════════════════════════════════════════════════════
  LOG OUTPUT SPECIFICATIONS
═══════════════════════════════════════════════════════════════════════════════

LOCATIONS:
  logs/pr_reviewer.log          ← Main log (DEBUG and above)
  logs/pr_reviewer_errors.log   ← Errors only (ERROR and above)

FORMAT:
  [timestamp] | [module:line] | [LEVEL] | [message]
  Example: 2026-07-23 14:32:15 | app.api.webhooks:45 | INFO | Processing PR...

COLORS:
  DEBUG      → Cyan      (\033[36m)
  INFO       → Green     (\033[32m)
  WARNING    → Yellow    (\033[33m)
  ERROR      → Red       (\033[31m)
  CRITICAL   → Red bg    (\033[41m)

ROTATION:
  Size: 10 MB per file
  Backups: 5 (total: 50 MB max)
  Policy: Automatic, no manual intervention needed

═══════════════════════════════════════════════════════════════════════════════
  KEY LOGGING PATTERNS
═══════════════════════════════════════════════════════════════════════════════

SUCCESS MARKERS:
  ✓ symbol marks all successful operations
  Examples:
    ✓ Webhook signature validation passed
    ✓ Installation token obtained (expires_at=...)
    ✓ Unified diff built: 7 files with patches, 1 skipped
    ✓ Security review complete: 2 findings (approved=false)
    ✓ Review posted to GitHub: review_id=5678910, 4 comments
    ✓ PR review complete and findings saved

FAILURE MARKERS:
  ✗ symbol marks all failures
  Includes error type and message
  Examples:
    ✗ FAILED to process PR (sha=...). Error type: HTTPStatusError
    ✗ GitHub API error exchanging JWT: status=401
    ✗ Private key file not found: ./github.pem

PR CONTEXT PREFIX:
  All PR-related logs include [owner/repo#number]
  Enables easy filtering: grep "owner/repo#15" logs/pr_reviewer.log
  Example: [owner/repo#15] Starting PR review pipeline

PHASE TRACKING:
  Each major phase: Starting... → ...processing... → ✓ ...complete
  Phases: webhook → auth → fetch → review (4 lenses) → aggregate → post

  COMMON TROUBLESHOOTING QUERIES
═══════════════════════════════════════════════════════════════════════════════

Find specific PR:          grep "owner/repo#15" logs/pr_reviewer.log
Find all failures:         grep "✗" logs/pr_reviewer.log
Find all successes:        grep "✓" logs/pr_reviewer.log
Find errors:               grep "ERROR\|CRITICAL" logs/pr_reviewer.log
Find warnings:             grep "WARNING" logs/pr_reviewer.log
Find GitHub API errors:    grep "GitHub API error" logs/pr_reviewer.log
Find auth failures:        grep "401\|403\|Invalid" logs/pr_reviewer.log
Find LLM errors:           grep "AuthenticationError\|RateLimitError" logs/pr_reviewer.log
Find skipped reviews:      grep "Skipping\|already reviewed" logs/pr_reviewer.log
